Weather Widget Integration Example

This page demonstrates how to easily integrate the weather widget into any webpage.

The weather widget is a self-contained component that can be added to any website with just a few lines of code.

How to integrate:

1. Include the CSS and JavaScript files:

<!-- Include the weather widget CSS --> <link rel="stylesheet" href="weather-widget.css"> <!-- Include the weather widget JavaScript --> <script src="weather-widget.js"></script>

2. Create a container element for the widget:

<div id="weather-widget-container"></div>

3. Initialize the widget with JavaScript:

<script> // Initialize the weather widget document.addEventListener('DOMContentLoaded', function() { const weatherWidget = new WeatherWidget('weather-widget-container', { apiKey: 'YOUR_API_KEY', // Optional: your own API key defaultCity: 'New York', // Optional: default city units: 'metric' // Optional: 'metric' or 'imperial' }); }); </script>
